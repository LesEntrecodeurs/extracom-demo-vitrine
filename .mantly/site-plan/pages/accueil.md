# Accueil

## Informations générales
- **Nom** : Accueil
- **Route** : /
- **Type** : editorial
- **Indexation** : index

## Objectifs
- Presenter Wedis, orienter vers le catalogue
- Expliquer le fonctionnement B2B
- Inciter l anonyme a se connecter ou s inscrire

## Structure
1. Hero : accroche + CTA Voir le catalogue + visuel
2. Onboarding visiteur : 3 etapes (si anonyme)
3. Mise en avant produits : produits en vedette
4. Categories : grille des familles
5. CTA final + FAQ

## Composants
- Hero (nouveau)
- InfoBanner (onboarding anonyme)
- FeaturedCarousel (produits)
- CategoryMenu (familles)
- ArticleCard
- JsonLd (WebSite, Organization, LocalBusiness)

## Style
Design Linear : epure, minimaliste, aere. Bleu/orange (logo).
