# Page 404 — Introuvable

## Informations générales
- **Nom** : Page introuvable
- **Route** : app/not-found.tsx (servie pour toute URL inconnue)
- **Type** : utilitaire
- **Indexation** : noindex (statut HTTP 404)

## Objectifs
- Rattraper une URL inexistante
- Ramener le visiteur vers l'accueil ou le catalogue

## Structure
1. Code 404 discret
2. Message « Page introuvable »
3. CTA : « Voir le catalogue » et « Accueil »
4. Champ de recherche (optionnel)

## Composants
- Bouton / lien (components/ui/button)
- EmptyState (optionnel)

## Style
Linear : bloc centré, sobre, cohérent avec le thème
