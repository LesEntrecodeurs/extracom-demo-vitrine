# Marques — Liste

## Informations générales
- **Nom** : Marques
- **Route** : /marques
- **Type** : editorial

## Objectifs
- Présenter les marques distribuées par Wédis (Kärcher, Nilfisk, Numatic, Viper, Tennant, etc.)

1. Titre H1 : « Les marques que nous distribuons »

3. Texte de présentation : Wédis est distributeur agréé de ces grandes marques
4. FAQ
